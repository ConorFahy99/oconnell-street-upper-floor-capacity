# QGIS project files

Save all .qgz project files here.

## Suggested file naming
- `oconnell_missing_floors.qgz`: main working project
- `oconnell_layout_A3.qgz`: print layout project (or keep layout within main project)

## Notes
- Set project CRS to EPSG:2157 (ITM) on creation.
- Store layer paths as relative paths (Project → Properties → General → Save paths: Relative).
  This keeps the project portable within the folder structure.
- Recommended plugins: QuickOSM, QuickMapServices (aerial imagery backdrop)
